    • Project Brief

    Battleship is a quantum version of the traditional board game that uses quantum data and concepts to enhance the gameplay. This project was created using the p5.party library by Justin Bakse to create a two-player game that can be player across computers and browsers. The code is open-source and the game can be played in the browser.

    Design Process

    The game has two main features: quantum maps and entangled ships.

    The maps were generated using a simple Bell state quantum algorithm programmed in Qiskit. The players ship placement is restricted by the map that changes every game and informs hitting strategy.

    Players can also choose to entangle two points on their own ships with those of the opponent. When these are hit they also destroy their enemy counterparts allowing for planned opponent sabotage.
